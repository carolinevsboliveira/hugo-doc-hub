## Quick Start

```bash
# 1. Copy .env.example to .env
cp .env.example .env

# 2. Run setup
bash scripts/setup.sh

# 3. Start local server
hugo server --buildDrafts

# Access: http://localhost:1313
```

## Stack

- **Hugo** — static site generation
- **Dochub Theme** — custom theme with full-text search
- **GitHub Actions** — automated build and deploy
- **Netlify** — hosting

## Key Features

✨ **Multi-language Support** — i18n natively built-in  
📚 **Documentation Management** — Technical, Product, and FAQ  
🤖 **Claude Integration** — Auto-generate docs with `/doc-pr`, `/doc-feature`, `/doc-module`  
🚀 **Automatic Deployment** — Push to main, auto-deploy to Netlify
