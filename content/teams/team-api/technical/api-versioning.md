## Estratégia de Versionamento

Mantemos compatibilidade com 2 versões principais de API simultaneamente.

### Versões Suportadas
- v3 (atual)
- v2 (deprecated)

### Migração
Sugerimos migração após 6 meses da release de nova versão.
